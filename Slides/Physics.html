Collision vs Physics

  • Collision is whether two objects intersect
  • Physics for our purposes is collision + applied force

Meshes

They are the objects that we will be applying physics on. You can think of them as sets of vertices, edges and faces.

Quick blender demo

Bounding Volumes

Simple shapes that completely contain an object. Used for optimizing collision detection. The most popular bounding volumes are bounding boxes, spheres and capsules.

Bounding Volume Hierarchies

A tree structure of bounding volumes. Can make the collision detection time complexity logarithmic in the number of objects.

Colliders 3D

  • Unitys implementation of bouning volumes
  • Invisible in Game View
  • GameObjects can have multiple colliders
  • Examples:
    • Box Collider
    • Sphere Collider
    • Capsule Collider

Colliders 2D

  • Are incompatible with 3D colliders
  • Work in a separate 2D Physics Engine
  • Examples:
    • Box Collider 2D
    • Circle Collider 2D
    • Capsule Collider 2D
    • Polygon Collider 2D

Triggers

By default colliders work for physics. To make them work just for collision we have to make them triggers. In scripts we work with them differently.

RigidBody(3D)

Component that enables physical behaviour for a GameObject

  • It makes it move according to the collisions and its physical properties
  • Allows affection by gravity
  • Add properties like Mass and Drag

RigidBody2D

  • Different Component from RigidBody
  • Works in a separate 2D Physics Engine

Static Colliders

When the GameObject doesn't have a RigidBody

  • Static Colliders should be used for immovable objects. Don't change their transform position because that could cause errors in the physics engine!
  • Example usage: Ground Terrain, Buildings, immovable objects...

Kinematic Rigidbody Collider

When the GameObject has a RigidBody with "is Kinematic" checked

  • Use Colliders with Kinematic RigidBody to make objects which you don't want to react to physics but still move them through scripts or something.
  • Example usage: Sliding Doors, Traps...
