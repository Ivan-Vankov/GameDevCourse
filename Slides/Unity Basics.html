MonoBehaviour

Base class for all Unity scripts. Determines what parts of the event loop the component will be in.

Most important methods/properties are:

  • Start() - called on object instantiation
  • Update() - called once per frame
  • gameObject - the GameObject this component is attached to
  • transform - the Transform attached to this GameObject
